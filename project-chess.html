♟️ Chess & Checkers Set

Arduino Chess Timer + Custom Board + 3D-Printed Pieces

Overview

This ongoing project combines electronics, CAD, and digital fabrication to create a fully custom chess set with an integrated Arduino-powered timer. The set includes a laser-cut chessboard, 3D-printed pieces, and a custom-designed case for the timer, with an emphasis on clean, minimalist design.

🧠 Features

  • ✅ Arduino Uno-based digital chess timer with OLED display
  • ✅ 3D-printed enclosure with diagonal vent slits for cooling
  • ✅ Full set of 3D-printed chess pieces (PLA) and optional checkers set
  • ✅ Laser-cut chessboard made from wood or acrylic
  • ✅ Minimalist internal layout for easy assembly

🛠️ Project Components

Electronics

  • Arduino Uno (or Nano)
  • 0.96" OLED Display (SSD1306 I2C)
  • 2× Momentary Push Buttons
  • Jumper Wires (M–F)
  • Power via USB or battery

Mechanical

  • case.STL – Timer enclosure
  • chess_pieces/ – Full STL set: pawn, rook, bishop, knight, queen, king
  • chessboard.dxf – Laser-cuttable 8×8 grid board
  • Optional: Screws or adhesive for Arduino mounting

📈 Progress

Currently prototyping the electronics and refining the enclosure design for a better fit with the Arduino components. 3D-printed chess pieces have reached a 93.75% print success rate after multiple design optimizations. Next steps include laser-cutting the frame and finalizing assembly.
